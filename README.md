# Python Expense Tracker

## Project Overview

Expense Tracker is a Python command-line application that helps users record and manage daily expenses.

The application allows users to:

- Add Expenses
- View Expenses
- Calculate Total Spending
- Find Highest Expense
- Find Lowest Expense
- Calculate Average Expense
